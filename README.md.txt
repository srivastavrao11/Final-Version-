# Blog Management System

## Project Overview

This is a PHP and MySQL-based Blog Management System developed as part of a Web Development Internship. The application allows users to register, log in, and manage blog posts securely.

## Features

* User Registration and Login
* CRUD Operations (Create, Read, Update, Delete)
* Search Posts
* Pagination
* Secure Authentication
* Prepared Statements (SQL Injection Prevention)
* Server-side and Client-side Form Validation
* Role-Based Access Control (Admin & Editor)
* Responsive User Interface using Bootstrap

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* Bootstrap
* XAMPP

## How to Run

1. Start Apache and MySQL using XAMPP.
2. Import the database into phpMyAdmin.
3. Place the project folder inside the `htdocs` directory.
4. Open the project in your browser using:
   `http://localhost/your-project-folder`
