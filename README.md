# Blog Management System

## Overview

The Blog Management System is a web application developed using PHP and MySQL. It allows users to register, log in, and manage blog posts through a secure and user-friendly interface. The project integrates essential web development concepts along with security enhancements to provide a reliable blogging platform.

## Features

* User Registration and Login
* Create, Read, Update, and Delete (CRUD) Operations
* Search Posts by Title or Content
* Pagination for Better Navigation
* Password Hashing for Secure Authentication
* Prepared Statements to Prevent SQL Injection
* Server-side and Client-side Form Validation
* Role-Based Access Control (Admin & Editor)
* Responsive User Interface using Bootstrap

## Technologies Used

## Installation

1. Install XAMPP and start Apache and MySQL.
2. Import the project database into phpMyAdmin.
3. Copy the project folder into the `htdocs` directory.
4. Open the application in your browser:
   `http://localhost/your-project-folder`

## Security Features

* Prepared statements for all database queries.
* Password hashing using PHP's `password_hash()` and `password_verify()`.
* Server-side and client-side form validation.
* Session-based authentication.
* Role-based authorization for protected actions.
